# AI task scheduling app (WIP)

## Tools being used
- React (frontend)
- Express (web framework for Nodejs)
- tRPC (type-safe API)
- Prisma (modern ORM for database)
- Postgres (database)
- TailwindCSS (styling)
- Typescript
- Vite (build tool)
- Vitest (testing)
- Turborepo (build-system for monorepo)
- Auth0 (authentication)
